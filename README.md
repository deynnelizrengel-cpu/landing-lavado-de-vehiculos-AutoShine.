AutoShine

## Descripción del Proyecto
AutoShine es una landing page desarrollada para promocionar servicios de lavado y detallado de vehículos. 
La plataforma permite mostrar los servicios disponibles, los planes de membresía y un formulario para solicitar información o realizar reservas de manera sencilla.

## Tema Seleccionado
Landing Page para un negocio de lavado de vehículos y servicios de estética automotriz.

## Herramientas Utilizadas
- HTML5
- CSS3
- JavaScript
- GitHub
- GitHub Pages

## Funcionalidad Principal
- Mostrar información sobre los servicios de lavado de vehículos.
- Presentar planes de membresía para clientes frecuentes.
- Permitir el registro de solicitudes y reservas mediante un formulario.
- Validar campos obligatorios (nombre y correo electrónico).
- Mostrar un mensaje de confirmación cuando la solicitud es enviada correctamente.
- Diseño responsive adaptable a dispositivos móviles y de escritorio.
